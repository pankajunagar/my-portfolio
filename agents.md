# Agentic Portfolio Architecture

This portfolio is built using a modern, performance-first stack designed for high-end user experiences.

## Technical Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theme**: Next-Themes (Dark/Light mode)

## Design System
The project follows a "Glassmorphic" design language:
- **Translucent Layers**: Heavy use of backdrop-blur and semi-transparent backgrounds.
- **Vibrant Accents**: Primary blue (#038ec7) used for focus points and gradients.
- **Dynamic Icons**: Standardized set of professional icons for better cognitive mapping.

## Animated Interactions
- **Staggered Reveals**: Scroll-triggered entry animations for all sections.
- **Micro-interactions**: Hover transforms, glow effects, and interactive SVG backgrounds.
- **Theme Transitions**: Smooth opacity and color shifts during theme switching.
